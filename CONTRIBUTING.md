# Как внести вклад в Flame

Спасибо за желание помочь проекту!

## Правила
1. Делай форк репозитория
2. Создавай ветку от main: `git checkout -b feature/название`
3. Следуй стилю кода (ESLint / Prettier / PEP8)
4. Проверяй тесты перед коммитом
5. Делай Pull Request с понятным описанием

## Обсуждение идей
Заведи Issue с тегом `enhancement`, обсудим перед разработкой.

## Сообщения о багах
Укажи: версию ОС, шаги для воспроизведения, скриншот.
